# CV Project Analysis

## Project Overview
This is a personal CV (Curriculum Vitae) website for Simon Berlendis, a software developer with a background in particle physics.

## Project Structure

```
CV/
├── index.html          # Main HTML file
└── styles.css          # CSS styling
```

## Key Components

### HTML Structure (index.html)
- **Header Section**: Contains name, title, and contact information
- **Main Content**: Two-column layout (left for main content, right for supplementary info)
- **Sections**:
  - Summary
  - Professional Experience (with nested sub-experiences)
  - Education
  - Projects
  - Skills (categorized)
  - Languages
  - Certifications
  - Interests

### CSS Styling (styles.css)
- **Color Scheme**: Primary burgundy (#722F37), secondary colors, gold accents (#c9a962)
- **Typography**: Uses Palatino Linotype font family
- **Layout**: CSS Grid for main content, Flexbox for components
- **Responsive Design**: Media queries for mobile and print views
- **Visual Elements**: Borders, icons, decorative accents, consistent spacing

## Technical Features

1. **Responsive Design**:
   - Desktop: Two-column layout
   - Mobile: Single column layout
   - Print: Optimized for printing

2. **CSS Features**:
   - CSS variables for theming
   - Flexbox and CSS Grid layouts
   - Media queries for responsiveness
   - Print-specific styling

3. **Content Organization**:
   - Professional experience with nested sub-experiences
   - Skills categorized by type
   - Certifications and publications listed
   - Bilingual language skills

## Content Highlights

- **Technical Skills**: Java/Kotlin, C#/.Net, Python, JavaScript/TypeScript
